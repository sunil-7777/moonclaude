Installation & Setup Guide

1. Install Claude Code CLI

MoonClaude connects to the official Anthropic Claude CLI. Make sure you have Node.js installed, then install Claude globally: Official Anthropic Docs

$ npm install -g @anthropic-ai/claude-code

2. Install MoonClaude

MoonClaude is deployed as a Python package. We highly recommend using pipx or global pip.

$ pip install moonclaude

3. Setup & Launch

Follow the interactive CLI wizard, then drop right into your workflow.

# Configure models, API keys, and memory $ moon setup
# Start the proxy and open Claude seamlessly $ moon launch

Bypassing Claude Restrictions

Authenticating Successfully

Claude Code by default only supports Anthropic models and requires an active initial authentication to their servers. To use MoonClaude, you must authenticate once via the official CLI:

$ claude login

Follow the browser prompt to log into your Anthropic Developer account. Don't worry—MoonClaude will route the actual requests away from Anthropic to your selected free providers instead.

Routing Through MoonClaude

Never run claude directly if you want it to be free. Always launch your environment with:

$ moon chat # OR $ moon launch

MoonClaude sets local HTTP proxy environmental variables (under the hood) to seamlessly override Anthropic's endpoints. Your prompts instantly hit Gemini, Llama, or Qwen instances instead securely!
